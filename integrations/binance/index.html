    🪙 바이낸스 (Binance) 연동 가이드

    세계 최대 암호화폐 거래소 중 하나인 바이낸스와의 연동을 통해 실시간 시세 수신 및 주문 실행을 지원합니다.

    📦 설치

    다음 명령어로 바이낸스 연동 기능을 포함하여 설치할 수 있습니다.

    pip install --upgrade "nautilus_trader[binance]"

    🔗 예제 코드: GitHub 예제 디렉토리에서 실제 작동하는 라이브 트레이딩 예제 스크립트를 찾을 수 있습니다.

    📝 개요 및 지원 상품

    이 어댑터는 바이낸스의 다양한 상품에 대한 라이브 시세 수신과 주문 실행을 지원합니다.

    상품 유형 지원 여부 비고
    현물 (Spot) ✅ 바이낸스 US 포함
    USDT-마진 선물 (무기한/인도) ✅
    COIN-마진 선물 ✅
    마진 (Margin) ❌ 현재 마진 거래는 구현되지 않음

    심볼 규칙

    바이낸스는 현물과 무기한 선물의 심볼이 BTCUSDT로 동일할 수 있습니다. NautilusTrader는 이를 명확히 구분하기 위해 모든 무기한 선물 심볼에 -PERP 접미사를 붙입니다. (예: BTCUSDT-PERP)

    ⚙️ 설정 방법

    라이브 트레이딩 노드(TradingNode)에 바이낸스 클라이언트를 설정하는 것이 가장 일반적인 사용 사례입니다. TradingNodeConfig에 다음과 같이 설정합니다.

    from nautilus_trader.adapters.binance import BINANCE
from nautilus_trader.live.node import TradingNodeConfig

config = TradingNodeConfig(
    # ... 다른 설정 생략 ...
    data_clients={
        BINANCE: {
            "api_key": "YOUR_BINANCE_API_KEY",
            "api_secret": "YOUR_BINANCE_API_SECRET",
            "account_type": "usdt_future",  # spot, usdt_future, coin_future 중 선택
            "testnet": False,  # 테스트넷 사용 시 True
        },
    },
    exec_clients={
        BINANCE: {
            "api_key": "YOUR_BINANCE_API_KEY",
            "api_secret": "YOUR_BINANCE_API_SECRET",
            "account_type": "usdt_future",
            "testnet": False,
        },
    },
)

    💡 팁: API 키와 시크릿은 코드에 직접 작성하는 대신, BINANCE_API_KEY, BINANCE_API_SECRET 같은 환경 변수를 사용하여 관리하는 것이 더 안전하고 편리합니다.

    📈 주요 기능 및 주문

    • 다양한 주문 유형: MARKET, LIMIT, STOP_LIMIT, TRAILING_STOP_MARKET 등 바이낸스가 지원하는 대부분의 주문 유형을 사용할 수 있습니다.
    • 고급 주문 기능: post_only(메이커 주문), reduce_only(포지션 감소 전용), OCO, 아이스버그 등 고급 주문 실행 옵션을 지원합니다.
    • 선물 헷지 모드 (Hedge Mode): 동일 상품에 대해 롱/숏 포지션을 동시에 보유할 수 있습니다. 사용하려면 바이낸스 계정 설정 변경, use_reduce_only=False 옵션 설정, 그리고 주문 시 position_id에 -LONG 또는 -SHORT 접미사를 붙여 방향을 명시해야 합니다.

    바이낸스 전용 데이터 구독

    선물 시장가(Mark Price)나 펀딩비(Funding Rate)와 같은 바이낸스 전용 데이터 스트림을 구독하여 전략에 활용할 수 있습니다.

    # on_start 핸들러 내부에서
from nautilus_trader.adapters.binance import BinanceFuturesMarkPriceUpdate
from nautilus_trader.model import DataType, ClientId

self.subscribe_data(
    data_type=DataType(BinanceFuturesMarkPriceUpdate, metadata={"instrument_id": self.instrument.id}),
    client_id=ClientId("BINANCE"),
)
