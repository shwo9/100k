    👨‍💻 개발자 가이드

    NautilusTrader 개발에 참여하거나 자신의 필요에 맞게 확장하고 싶은 분들을 위한 가이드입니다.

    🎯 개발 철학

    NautilusTrader의 핵심 개발 철학은 "업무에 적합한 최고의 도구를 사용한다" 입니다. 이를 위해 다음과 같은 접근 방식을 취합니다.

    • Python의 장점 활용: Python의 높은 생산성과 풍부한 라이브러리 생태계를 최대한 활용합니다.
    • Python의 단점 극복: 인터프리터 언어인 Python의 고질적인 문제인 성능과 타입 안전성 부족을 Cython과 Rust를 통해 극복합니다.

    Cython은 Python 코드를 C 코드로 변환하여 컴파일된 확장 모듈을 생성합니다. 이를 통해 Python의 단순함과 C의 네이티브에 가까운 성능을 동시에 달성할 수 있습니다.

    💡 개발에 참여하기 위해 C 언어 전문가가 될 필요는 없습니다. 하지만 Cython 공식 문서를 통해 Cython이 C 타입을 어떻게 사용하는지 기본적인 개념을 익히는 것이 큰 도움이 됩니다.
