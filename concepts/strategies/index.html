    🧠 전략 (Strategies)

    NautilusTrader의 사용자 경험의 핵심은 트레이딩 전략을 작성하고 운영하는 것입니다. 이 가이드에서는 전략을 구성하는 방법과 주요 기능을 설명합니다.

    🏛️ 전략의 기본 구조

    모든 트레이딩 전략은 Strategy 클래스를 상속받아 구현합니다. 전략은 시스템의 핵심 행위자인 Actor의 모든 기능을 가지며, 추가로 주문 관리 기능을 갖습니다.

    전략은 크게 두 부분으로 나뉩니다:

    1. 전략 구현 클래스: Strategy를 상속받아 실제 매매 로직을 구현합니다.
    2. 전략 설정 클래스 (선택 사항): StrategyConfig를 상속받아 전략에 필요한 파라미터(진입 수량, 사용할 상품 등)를 정의합니다. 설정을 분리하면 동일한 코드를 백테스트와 라이브 트레이딩에 그대로 사용할 수 있어 매우 효율적입니다.

    📞 핸들러 (Handlers)

    전략의 로직은 on_* 형태의 핸들러 메서드 안에 작성합니다. 시스템은 특정 이벤트가 발생하면 해당하는 핸들러를 자동으로 호출합니다.

    • 상태 핸들러: on_start() (전략 시작 시 초기화), on_stop() (전략 종료 시 정리) 등 생명주기 관련 작업을 수행합니다.
    • 데이터 핸들러: on_bar(), on_quote_tick() 등 새로운 시장 데이터가 수신될 때마다 호출되어 주된 분석과 판단을 수행합니다.
    • 주문/포지션 핸들러: on_order_filled() (주문 체결 시), on_position_opened() (포지션 진입 시) 등 주문 및 포지션 상태 변경에 대응합니다.
    # on_start 핸들러 예시
def on_start(self) -> None:
    # 사용할 상품 정보 로드
    self.instrument = self.cache.instrument(self.config.instrument_id)

    # EMA 지표 등록
    self.register_indicator_for_bars(self.config.bar_type, self.fast_ema)
    self.register_indicator_for_bars(self.config.bar_type, self.slow_ema)

    # 과거 데이터 요청 (지표 초기화용)
    self.request_bars(self.config.bar_type)

    # 실시간 데이터 구독
    self.subscribe_bars(self.config.bar_type)

    🛠️ 전략의 주요 기능

    시계와 타이머

    self.clock을 통해 현재 시간을 조회하거나, 특정 시간에 또는 주기적으로 로직을 실행시키는 시간 알림(set_time_alert)과 타이머(set_timer)를 설정할 수 있습니다.

    캐시와 포트폴리오 접근

    • 캐시 (self.cache): 현재가, 체결가, 바 데이터는 물론, 주문과 포지션 상태 등 시스템의 거의 모든 정보에 빠르게 접근할 수 있습니다.
    • 포트폴리오 (self.portfolio): 계좌의 잔고, 미실현/실현 손익, 순포지션 등 포트폴리오와 관련된 정보를 실시간으로 조회할 수 있습니다.

    거래 명령어

    • 주문 제출: self.order_factory로 MARKET, LIMIT 등 다양한 주문 객체를 생성하고, self.submit_order()로 제출합니다.
    • 주문 취소: self.cancel_order(), self.cancel_orders() 등으로 개별 또는 전체 주문을 취소합니다.
    • 주문 정정: self.modify_order()로 주문의 수량이나 가격 등을 변경합니다.

    ⚙️ 전략 설정 (StrategyConfig)

    전략에 필요한 파라미터(예: instrument_id, trade_size 등)를 별도의 설정 클래스로 분리하면 코드의 재사용성과 유연성이 크게 향상됩니다. 설정값은 전략 내에서 self.config를 통해 접근할 수 있습니다.

    # 설정 클래스 정의
class MyStrategyConfig(StrategyConfig):
    instrument_id: InstrumentId
    fast_ema_period: int = 10
    slow_ema_period: int = 20
    trade_size: Decimal

# 전략 클래스 정의
class MyStrategy(Strategy):
    def __init__(self, config: MyStrategyConfig) -> None:
        super().__init__(config) # 설정과 함께 부모 클래스 초기화

    def on_bar(self, bar: Bar):

            size = self.config.trade_size
