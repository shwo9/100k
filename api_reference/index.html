    🐍 Python API 레퍼런스

    NautilusTrader의 Python API 레퍼런스에 오신 것을 환영합니다!

    이 문서는 NautilusTrader 프레임워크를 구성하는 모든 모듈, 클래스, 메서드, 함수에 대한 상세한 기술 정보를 제공합니다. 모든 내용은 최신 소스 코드의 docstring을 기반으로 Sphinx를 통해 자동으로 생성됩니다.

    📣 버전 안내

    API 레퍼런스는 두 가지 버전으로 제공됩니다. 좌측 상단의 Versions 드롭다운 메뉴에서 원하는 버전을 선택할 수 있습니다.

    • latest: 최신 안정 릴리즈 버전에 대한 문서입니다.
    • nightly: 최신 실험적 기능이 포함된 개발 버전에 대한 문서입니다.

    우측의 탐색 사이드바를 사용하여 원하는 모듈을 찾아보고, 각 항목을 클릭하여 파라미터, 반환값 등에 대한 상세한 설명을 확인할 수 있습니다.

    🧩 모듈 요약

    NautilusTrader는 기능별로 모듈화되어 있습니다. 각 모듈의 주요 역할은 다음과 같습니다.

    • core: 프레임워크 전반에서 사용되는 저수준 핵심 컴포넌트, 상수, 함수들을 포함합니다.
    • common: 다양한 컴포넌트를 조립하는 데 공통적으로 사용되는 부분들을 포함합니다.
    • model: Instrument, Order, Position, Account 등 트레이딩 도메인을 나타내는 풍부한 데이터 모델을 정의합니다.
    • config: Strategy, BacktestNode, TradingNode 등 시스템의 여러 컴포G넌트를 설정하기 위한 클래스들을 제공합니다.
    • trading: Strategy 클래스를 포함하여 트레이딩 로직과 관련된 컴포넌트 및 도구를 제공합니다.
    • data: 데이터 처리 스택과 Bar, QuoteTick 등 데이터 관련 클래스들을 포함합니다.
    • persistence: 데이터 저장, 카탈로깅, 조회를 담당하며, 주로 백테스팅을 지원합니다. ParquetDataCatalog가 핵심입니다.
    • cache: 고성능 인메모리 캐싱 인프라를 제공하여 데이터와 객체에 빠르게 접근할 수 있도록 합니다.
    • execution: 주문 실행 스택을 담당하며, 주문의 생명주기를 관리합니다.
    • portfolio: 포트폴리오 및 계좌 관리 기능을 제공합니다.
    • accounting: 다양한 계좌 유형(현금, 마진 등)과 계좌 관리 메커니즘을 다룹니다.
    • risk: 리스크 관리 컴포넌트 및 도구를 제공합니다.
    • analysis: 트레이딩 성과 통계 및 분석과 관련된 컴포G넌트를 포함합니다.
    • indicators: 이동평균, RSI 등 효율적인 기술적 분석 지표들을 제공합니다.
    • backtest: 백테스트 엔진 및 노드 구현 등 백테스팅과 관련된 컴포넌트를 포함합니다.
    • live: 라이브 트레이딩 엔진 및 클라이언트 구현 등 라이브 트레이딩 관련 컴포넌트를 포함합니다.
    • system: backtest, sandbox, live 환경 전반에 걸쳐 공통적으로 사용되는 핵심 시스템 커널을 제공합니다.
    • adapters: 외부 거래소 및 데이터 소스와의 연동을 위한 어댑터들을 포함합니다.
    • serialization: 객체 직렬화/역직렬화를 위한 기본 컴포넌트와 구현체들을 포함합니다.

    🤔 왜 Python인가?

    Python은 간결하고 직관적인 문법으로 시작하여, 현재는 세계에서 가장 인기 있는 범용 객체 지향 프로그래밍 언어로 성장했습니다. 특히 데이터 과학, 머신러닝, AI 분야에서는 사실상의 표준 언어(lingua franca)로 자리 잡았습니다.

    물론 Python은 인터프리터 언어로서 성능에 한계가 있지만, NautilusTrader는 Cython을 통해 이 문제를 해결했습니다. Cython은 Python에 정적 타이핑과 C언어 수준의 성능을 더해주어, Python의 풍부한 생태계와 고성능 시스템의 장점을 모두 누릴 수 있게 해줍니다.
